

Pattern
-------

Object of this program is that from a given pattern of letters, the common set of letters 
have to be compressed and the compressed string must be displayed, so that when the compressed 
string is expanded the given pattern should reappear. The sets may contain single or multiple letters.

The format of the input and output is shown in below example.

Ex: Sample input: abcabcaaaaababab
          Output: k*3l*4m*3,k=abc,l=a,m=ab
