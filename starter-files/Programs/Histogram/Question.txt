

Histogram
----------

Object of this program is to read data from a file and find out the count of different attributes.
The format of the data is in the sample file attached in this folder.
Sample file of data is attached in this folder where you would find the different attributes.

The input will be the file and the output should be displayed with attribute name and the
count of that attribute.


Ex: Input: file of data
    Output: city:5
