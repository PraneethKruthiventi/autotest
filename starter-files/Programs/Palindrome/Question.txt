

Palindrome
----------

Object of this program is to find out if a sentence or a word is a valid palindrome
or not while ignoring all the case of letters(a or A) full stops, question marks, 
exclamatary marks, spaces, etc.

Ex:  Sample Input: Never odd or even.   
           Output: Yes

           Input: Malayalam
           Output: yes

           Input: Kansas
          Output: no
